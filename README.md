# README

### Oficina 'Testando Rails Requests'

## Requeriments

* Ruby version: 2.5.1

* Rails version: 5.2.8

## Set up the app

After downloading the repository:

    $ cd lista-telefonica
    $ bundle install
    $ rails db:create db:migrate
    $ rails s


## To run specs

    $ rspec ./spec/path_to_test
